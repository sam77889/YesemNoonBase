# 📦 NOON Base - 跨境电商洞察与智能爬虫系统

## 📑 项目概述
NOON Base 是一个针对中东电商巨头 NOON 平台量身定制的**一站式数据抓取、清洗与可视化大盘系统**。
项目采用前后端分离架构，解决了 NOON 平台因架构升级（Next.js -> TanStack Router）以及强制防爬策略带来的抓取难题，提供从“关键词发起任务”到“落库追踪”，再到“大盘多维可视化”的闭环体验。

---

## 🛠️ 技术栈架构
* **前端 (Frontend)**: React 18 + TypeScript + Vite, UI 采用现代化玻璃拟态 (Glassmorphism) 与 Bento Grid 布局。
  * **图表库**: Recharts (实现了散点图、面积图、多轴组合图、横向柱状图等复杂图表)
  * **动画**: Framer Motion
* **后端 (Backend)**: FastAPI (Python) + SQLite (轻量级本地存储)
  * **异步并发**: `asyncio` 控制爬虫并发量，保障请求稳定性。
  * **ORM**: SQLAlchemy (异步引擎支持)
  * **网络代理**: 深度整合动态代理 (支持 ScraperAPI Premium / Oxylabs)

## 🚀 核心功能模块

### 1. 爬虫控制中心 (Scraper Engine)
* **TSR 深度嗅探技术**：攻克了 NOON 放弃传统 DOM 渲染的防爬机制。在爬虫引擎 (`scraperapi.py`) 中，创新性地使用正则提取页面底层的 `__TSR__` 序列化状态树，无需启动沉重的无头浏览器即可闪电般精准提取包含**评论数 (Review Count)**、**评分 (Rating)** 在内的深层动态数据。
* **实时终端日志展现**：前端实现了打字机效果和平滑滚动的模拟终端，后端爬取日志会实时展示，使用户明确知晓“正在并发请求”、“正在解析数据”、“已入库数量”。

### 2. 追踪商品库 (Database Tracker)
* **软删除机制**：前端支持批量剔除无效竞品，后端采用 `status = 'INACTIVE'` 的软删除逻辑，保证数据可追溯，同时确保被剔除的数据不会污染大盘统计。
* **容量突破**：支持千级别的大数据量前端承载与动态过滤渲染。

### 3. 市场分析大盘 (Market Analytics Dashboard)
打破传统死板的数据罗列，提供极具洞察力的 4 大核心可视化图表，并支持**按类目 (Category) 实时动态过滤联动**：
* **💰 蓝海价格段分析 (组合图)**：价格区间商品数 (竞争热度) vs 总评论数 (需求热度)。
* **📈 预估销量大盘分布 (面积图)**：基于独创的“伪造销量公式”（评论数*转化率+低价加权+评分加权），以紫色平滑面积图直观展示市场是“赢家通吃”还是“长尾健康市场”。
* **🎯 价格 vs 评论数 (散点图)**：X轴价格，Y轴评论数，气泡大小为评分。一眼定位“左上角”的高评价低价蓝海爆款。
* **🏆 品牌竞争力 TOP10 (横向柱状图)**：基于总评论声量排行，附带单品数量、均分展示，清晰揭露品类垄断格局。

## 💻 本地部署与运行指南

**1. 启动后端 API 服务**
```bash
cd noon_api
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

**2. 启动前端监控面板**
```bash
cd noon_dashboard
npm install
npm run dev
# 默认运行在 http://localhost:5173 
```
